# Commonly available packages

The following packages are commonly available to install. Additional packages may be added over time.

| Feature             | Package name               | Version      | Purpose                                                              |
|---------------------|----------------------------|--------------|----------------------------------------------------------------------|
| Data Models         | `cfs_base_data_models`     | `2.2.0`      | System Planner configuration                                         |
| MSDK                | `msdk`                     | `2.2.0`      | Develop MAX32xxx/MAX7800x projects                                   |
| Plugins             | `cfs_base_plugins`         | `2.2.0`      | Workspace creation, System Planner, code generation                  |
| Zephyr 4.3.0        | `zephyr`                   | `4.3.0`      | Zephyr 4.3.0 support for MAX32xxx/MAX7800x SoCs                      |

!!! note
    Only packages relevant to your permissions are displayed. If no packages are available or all are already installed, an information message will appear instead.
