# Project

The `project` command manages projects defined in a CFS workspace (`.cfsworkspace` file). It can be used to regenerate projects from the command line.

## Create project

```sh
cfsutil project create -w <path/.cfsworkspace> -p <project-name> [-s <plugin/search/directory>]
```

Regenerates a project defined within an existing `.cfsworkspace` file.

!!! important
    The workspace and its `.cfsworkspace` file must already exist before using this command. This command **regenerates** projects that are already defined in the `Projects` array. It cannot create new projects or add projects to an existing workspace—it only rebuilds existing projects, typically after a plugin upgrade or workspace update. The command **overwrites the project structure** based on the latest plugin templates.

    **Known limitation:** Workspaces created from templates may have an empty `Projects` array. **Because this command only regenerates existing projects, it will fail if the `Projects` array is empty.** To use `cfsutil project create`, ensure your workspace has a populated `Projects` array by using the [**Manually configure the workspace**](../workspaces/create-new-workspace.md#manually-configure-the-workspace) option in the Workspace Creation Wizard or by running `cfsutil workspace configure` followed by `cfsutil workspace create -i`. See [Known issues](../../release-notes/2.2.0.md#cfs-command-line-cfsutil) for details.

Refer to the example below. The `-p` flag must match the **Name** value of the project you want to generate.

| Flag           | Description           |
|----------------|-----------------------|
| `-p=<name>`    | **Required.** Name of the project to generate. Must match a **Name** field in the JSON. If the name contains spaces or special characters (for example, `ARM Cortex-M4F`), wrap it in quotes. |
| `-s=<path>`    | (Optional) Additional directory paths to search for plugins and data models. Can be used multiple times.|
| `-w=<path>`    | **Required.** Path and filename of the `.cfsworkspace` file |

!!! example

    ```sh
    cfsutil project create -w /path/to/workspace/Apard_ws/.cfs/.cfsworkspace -p "Arm Cortex-M4F"
    ```
