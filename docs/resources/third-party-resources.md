# Third party tools

## Olimex Arm Debugger

The Olimex ARM-USB-OCD-H debugger is required to debug the RISC-V core on supported MAX32xxx and MAX78xxx devices.

Download and installation instructions can be found in chapter 3 of the [Olimex ARM-USB-OCD-h User Manual](https://www.olimex.com/Products/ARM/JTAG/_resources/ARM-USB-OCD_and_OCD_H_manual.pdf)

## Segger J-Link Debugger

Segger's J-Link is an alternative debugger for ARM cores.

Download and installation instructions can be found on the [Segger website](https://www.segger.com/downloads/jlink/)

## Segger SystemView

Segger SystemView is available for free to Analog Devices customers for supported processors.

You must use SystemView version 3.58 or later.

To request a free license, visit
[Segger License Activation Request page](https://license.segger.com/adi.cgi)

Download and installation instructions can be found on the Segger website at
[Segger SystemView page](https://www.segger.com/products/development-tools/systemview/)
